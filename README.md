# TypeScript Console Application

Простое консольное приложение на TypeScript.

## Установка

```bash
npm install
```

## Использование

Для разработки:
```bash
npm run dev
```

Для сборки:
```bash
npm run build
```

Для запуска собранного приложения:
```bash
npm start
```

Для автоматической пересборки при изменении файлов:
```bash
npm run watch
```
